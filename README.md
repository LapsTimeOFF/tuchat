 # Instalation :
 
  ## Prérequis :
  - [Node.js](https://nodejs.org/) >= 10
  - npm >= 5

 ## Installation :
  Voici la liste des commandes à éxécuter :
    ```sh
    # Cloner le repo github
    git clone https://github.com/LapsTimeOFF/tuchat/
    # Ou télécharger le dernière Releases
    
    # Aller à l'interrieur du dossier
    cd tuchat
    
    # Installer
    npm install
    ```
